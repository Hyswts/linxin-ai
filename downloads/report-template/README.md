> 林昕AI · 免费资源 | 输入数据 → 自动生成专业Excel报表

---

## 这个模板能做什么

把你的销售数据（CSV/Excel）丢进去，一键生成包含**KPI看板、趋势图表、区域分析**的专业报表，告别手动拼Excel。

**生成的报表包含5个工作表：**

| 工作表 | 内容 |
|--------|------|
| 📊 数据总览 | KPI卡片 + 月度趋势折线图 + 产品占比饼图 |
| 📋 销售明细 | 全量数据，利润率自动标色（绿=高利润，红=低利润）|
| 🗺️ 区域分析 | 各区域销售额/利润对比柱状图 |
| 📦 产品分析 | 产品排名 + 销售额对比柱状图 |
| 📱 渠道分析 | 各渠道贡献占比 + 柱状图 |

## 快速开始（3步）

### 1. 安装Python（如果没有）

前往 [python.org](https://python.org) 下载安装 Python 3.8+，
安装时 **勾选 "Add to PATH"**。

### 2. 安装依赖

打开终端（Windows: CMD / macOS: Terminal），运行：

```bash
pip install pandas openpyxl
```

### 3. 生成报表

**用样例数据测试：**
```bash
python generate_report.py
```

**用你自己的数据：**
```bash
python generate_report.py 你的数据.csv
```

运行后会在当前目录生成 `销售数据自动化报表.xlsx`。

## CSV数据格式

你的CSV文件需要包含以下列（第一行为表头）：

```
日期,产品,区域,渠道,客户类型,单价,数量,销售额,成本,利润
2026-01-05,蓝牙耳机,华东,抖音,新客,129,15,1935,871,1064
2026-01-08,智能手表,华南,淘宝,老客,399,8,3192,1277,1915
```

| 列名 | 类型 | 说明 | 示例 |
|------|------|------|------|
| 日期 | 日期 | 格式 YYYY-MM-DD | 2026-01-05 |
| 产品 | 文本 | 产品名称 | 蓝牙耳机 |
| 区域 | 文本 | 销售区域 | 华东 |
| 渠道 | 文本 | 销售渠道 | 抖音 |
| 客户类型 | 文本 | 新客/老客/VIP | 新客 |
| 单价 | 数字 | 单件价格 | 129 |
| 数量 | 数字 | 购买数量 | 15 |
| 销售额 | 数字 | 单价 × 数量 | 1935 |
| 成本 | 数字 | 总成本 | 871 |
| 利润 | 数字 | 销售额 - 成本 | 1064 |

> 💡 **最小数据集**：至少需要日期、产品、销售额3列即可运行，其他列可选。

## 自定义

### 更换配色

编辑 `generate_report.py` 顶部的 `COLORS` 字典：

```python
COLORS = {
    "green": "22C55E",      # 主色
    "green_dark": "15803D",  # 深色
    ...
}
```

### 增加分析维度

在 `create_report()` 函数中参照现有Sheet的写法，添加新的分析工作表。

## 常见问题

**Q: 报错 `No module named 'pandas'`**
A: 运行 `pip install pandas openpyxl` 安装依赖。

**Q: 图表显示为空白**
A: 确保CSV中的数值列是数字格式，不要有 ¥ 符号或逗号。

**Q: 想增加更多图表**
A: 参考 `generate_report.py` 中的 `BarChart` / `PieChart` 代码段，openpyxl文档：https://openpyxl.readthedocs.io/
