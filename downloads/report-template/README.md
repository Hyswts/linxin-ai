# 📊 数据自动化报表模板

> 林昕AI — 企业AI自动化服务 | 免费资源

## 这是什么

一个Python脚本，输入你的销售数据（CSV/Excel），自动生成一份**专业格式化的Excel报表**，包含：

- 📊 **数据总览** — KPI卡片 + 月度趋势 + 产品占比饼图
- 📋 **销售明细** — 全量数据，利润率自动标色
- 🗺️ **区域分析** — 各区域销售对比柱状图
- 📦 **产品分析** — 产品排名 + 销售额对比
- 📱 **渠道分析** — 各渠道贡献占比

## 快速开始

```bash
# 1. 安装依赖
pip install pandas openpyxl

# 2. 用样例数据运行
python generate_report.py

# 3. 用你自己的数据
python generate_report.py your_data.csv
```

## CSV数据格式

你的CSV文件需要包含以下列：

| 列名 | 类型 | 说明 |
|------|------|------|
| 日期 | 日期 | 格式: YYYY-MM-DD |
| 产品 | 文本 | 产品名称 |
| 区域 | 文本 | 销售区域 |
| 渠道 | 文本 | 销售渠道 |
| 客户类型 | 文本 | 新客/老客/VIP |
| 单价 | 数字 | 单件价格 |
| 数量 | 数字 | 购买数量 |
| 销售额 | 数字 | 单价×数量 |
| 成本 | 数字 | 总成本 |
| 利润 | 数字 | 销售额-成本 |

## 输出效果

运行后生成 `销售数据自动化报表.xlsx`，包含：
- 专业配色（绿色系，与林昕AI品牌一致）
- 自动计算的KPI指标
- 内嵌图表（折线图、柱状图、饼图）
- 条件格式（利润率高亮）
- 交替行背景色

## 自定义

修改 `generate_report.py` 中的 `COLORS` 字典可以更换配色方案。
